# React Metronome App

Source application for the React Metronome.

The root README contains the main project description, live app link, and deployment information.

## Scripts

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Create a production build:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Notes

This application was originally created with Create React App and later refreshed with a more modern component structure, updated UI, and cleaner project documentation.
